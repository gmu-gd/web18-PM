Instructions

The Orignal File

  1. Open Illustrator
  2. Select the rectangle tool
  3. Using the rectangle tool, make a 1.3in x 1.3in box, with the default stroke and fill colors.
  4. Make another box using the rectangle tool, with 2.8in x 1.5in dimensions.
  5. Move the boxes so that the bottom right corner of the smaller one connects to the top left of the bigger one.
  6. 6. Select the “Line Segment Tool”
  7. Create two horizant line segments that are 1.3in long.
  8. In the angle dropdown menu, change the degree of one line to 240* and the other to 120*.
  9. Move the two lines together so that their ends touch and the lines resemble and upside down v.
  10. Select both of the lines, and moves them so that the point of that upside down v falls on the bottom left corner of the bigger rectangle box.
  11. Copy and paste the two lines, and move the new set to the bottom right corner of the bigger ractangle.
  12. Select all of the created objects and group them together.
  13. Select the current artboard and change it to a 7in. x 5in. and move the object group so it falls in the center of the page.
  14. Export as a jpg and send to iyoung@gmu.edu.
